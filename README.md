# Run this

Clone this
```
git clone git@github.com:JordyMoos/gameemperor-in-elm.git
```
```
cd gameemperor-in-elm
```

Install dependencies
```
bower install
yarn install
```

Run the app
```
yarn dev
```

Goto `localhost:8080` in your browser.
