# Run this

Clone this
```
git clone https://github.com/JordyMoos/game-site-in-elm.git
```
```
cd game-site-in-elm
```

Install dependencies
```
bower install
yarn install
```

Run the app
```
yarn dev
```

Goto `127.0.0.1:8888` in your browser.
