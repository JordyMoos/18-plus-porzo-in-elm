# Run this

Clone this
```
git clone git@github.com:JordyMoos/gameemperor-in-elm.git
```
```
cd gameemperor-in-elm
```

Run the dev env
```
elm-app start
```

Goto localhost:3000 in your browser.

**Make sure you have some CORS ignoring tool**
I use CORSe addon in firefox
